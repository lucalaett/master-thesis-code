# Master Thesis Code

Code and analysis notebooks for my master's thesis.

## Repository structure

* `notebooks/` – Jupyter notebooks used for analysis and figure generation
* `src/thesis/` – reusable Python functions for analysis and plotting
* `figures/` – exported figures used in the thesis

## Usage

Install the required Python packages (for example with pip or conda), then open the notebooks in the `notebooks/` directory and run the analysis cells to reproduce the figures.

## Notes

Notebooks contain the analysis workflow, while reusable functions are stored in `src/thesis/`.
